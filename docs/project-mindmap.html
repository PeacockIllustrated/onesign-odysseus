Central hub

Everything in this app orbits the canonical pipeline quote → artwork → production → delivery. Invoice branches off quote; purchase orders hang loose. Clients are a data substrate — never log in.

The inheritance chain

Every downstream record carries org_id, contact_id, site_id from the quote. Dashed teal lines show this — clients aren't visited, they're inherited.

Red edges = missing email

Token generation works end-to-end for /approve/artwork/[token] and /delivery/[token], but nobody sends the URL. Resend integration is the highest-value P0.

Amber edges = manual

Invoice creation and PO linkage are wired but not auto-triggered. Quote→invoice is a button; quote→PO is entirely manual navigation.

Spec-bearing row

artwork_component_items (satellite near ARTWORK) holds material, method, finish, dimensions, and target_stage_id. Release rebuilds job_items.stage_routing from these.

Legacy constellation

Bottom-left cluster shows orphan routes from the old multi-tenant SaaS life (/dashboard, /assets, /billing, /admin/orgs). Candidates for deletion.
